# ✅ To-Do 기능 기획서 (PRD)

## 1. 개요
사용자가 해야 할 일을 리스트에 추가하고, 조회하고, 삭제할 수 있는 간단한 To-Do 기능

## 2. 기능 상세

- ✅ 작업 추가 (`add_task`)
  - 입력한 텍스트를 리스트에 저장한다

- ✅ 작업 조회 (`list_tasks`)
  - 현재까지 추가된 작업 목록을 보여준다

- ✅ 작업 삭제 (`remove_task`)
  - 인덱스를 통해 특정 작업을 삭제한다

## 3. 고려사항

- 입력값 유효성 검증
- 중복 처리 여부는 논의 필요
- 데이터 저장은 우선 메모리 방식, 이후 DB 확장 가능성 고려
